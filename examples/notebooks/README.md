# Notebook Examples

This folder is reserved for developer and collaborator notebook examples.

Notebook-based validation, exploration, or demonstration workflows can be added
here in future pull requests when they provide value beyond the corresponding
Python scripts.

Current intent:

- keep the folder available as the canonical home for notebook examples
- avoid mixing notebooks into the top level of `examples/`
- allow future `.py` and notebook counterparts to coexist with a clear layout
